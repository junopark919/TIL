# The Basic Command Line

## ls(Mac) / dir(Windows)

See the contents of the folder.

```bash
ls
dir
```

## cd

Change the current directory.

```bash
cd <path>
```

## clear(Mac) / cls(Windows)

Clear up the current display.

```bash
clear
cls
```

## mkdir

Create a new folder.

```bash
mkdir <path>
```

## touch(Mac) / edit(Windows)

Change the current directory.

```bash
touch <filename>
edit <filename>
```

## cp(Mac) / copy(Windows)

Copy the file.

```bash
cp <filename> <path>
copy <filename> <path>
```

## mv(Mac) / move(Windows)

Move the file.

```bash
mv <filename> <path>
move <filename> <path>
```

## rm(Mac) / del(Windows)

Remove the file/folder. (Permanently, not into a trash bin)

- File

  ```bash
  rm <filename>
  del <filename>
  ```

- Folder

  ```bash
  rm -r <name>
  del <name>
  ```

## open(Mac) / ""(Windows)

Open a file up.

```bash
open <filename>
"path"
```
