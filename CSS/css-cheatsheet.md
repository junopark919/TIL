# CSS Cheatsheet

## clip-path

Make a complex shape by clipping an element

<br>

**Usage**

```css
clip-path: polygon(0 0, 100% 0, 100% 75vh, 0 100%);
```

<br>

## :where()

Apply some style to multiple elements at once.

<br>

**Usage**

```css
.card :where(h1, h2, h3) {
  color: #fafafa;
}
```
